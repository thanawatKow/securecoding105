Smashing ผู้โจมตีจะผ่าน malicious code และ ก๊อปปี้ของ buffers address ด้วย parameter เพื่อให้เกิดช่องโหว่ผ่าน argument p ฟังชั่น Strcpy จะเกิด overflow เมื่อมีการ return ที่อยู่ของฟั่งชั่นด้วยที่อยู่ของ buffer ซึ่งบรรจุ shell code การคืนค่าของชุดคำสั่งโดยใช้ที่อยู่ของเป้าหมาย buffer ที่จะ return address และ ไปทำ โค้ดที่ผู้โจมตีต้องการ
