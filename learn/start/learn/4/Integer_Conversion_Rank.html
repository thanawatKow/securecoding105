Buffer Overflow    
              buffer overflow คือการเขียนข้อมูลเกินขอบเขตที่ memory ที่ถูกกำหนดไว้ในโครงสร้างข้อมูลที่มีการจอง ไว้
สาเหตุที่ทำให้เกิด
        1)   Buffer overflow เกิดจาก เราไม่ได้ ตรวจขอบเขตของ array
        2)   Standard library functions ไม่ได้มีการตรวจขอบเขตของ array
        3)   Programmers ไม่ได้ตรวจขอบเขตของ array
*หมายเหตุไม่ใช่ทุก Buffer overflow จะเกิด ช่องโหว่ได้
